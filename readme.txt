PayPal yearly subscription is a very simple tool for creating restricted access to a user role by yearly subscription.

== Description ==

Subscriptions are always complicated. This plugin uses PayPal Express Checkout method for payment and uses shortcodes for managing the system.

All you have to do is insert the shortcodes in their rightful places, set up the paypal credentials in the setup page and you are ready to go.

== Installation ==


1. Upload the plugin folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Enter settings->PayPal yearly subsc. and fill out the settings.
4. Place the shortcodes into pages and test the system -> ready to go.
